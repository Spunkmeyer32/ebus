# Introduction

This library handles the communication with heating engineering via the eBUS specification. This protocol is used by many heating manufacturers in Europe.

_Notice: This is not an offical eBUS library!_

## Required

* slf4j
* commons-io
* commons-lang
* gson

* nrjavaserial
